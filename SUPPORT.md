## Reporting Issues

### Before Reporting

1. **Check Documentation**: Review the documentation and troubleshooting guides
2. **Search Issues**: Search existing issues to avoid duplicates
3. **Update**: Ensure you're using the latest version
4. **Reproduce**: Try to reproduce the issue consistently

### Bug Reports

When reporting a bug, please include:

- **Description**: Clear description of the issue
- **Steps to Reproduce**: Detailed steps to reproduce the problem
- **Expected Behavior**: What should happen
- **Actual Behavior**: What actually happens
- **Environment**:
  - Operating System and version
  - Python version (if running from source)
  - Count-Cups version
  - Camera model (if relevant)
- **Logs**: Relevant log files from `~/.count-cups/logs/`
- **Screenshots**: If applicable, include screenshots

### Feature Requests

When requesting a feature, please include:

- **Description**: Clear description of the feature
- **Use Case**: Why is this feature needed?
- **Proposed Solution**: How should this feature work?
- **Alternatives**: What alternatives have you considered?

## Common Issues

### Camera Not Detected

- Check camera permissions in system settings
- Try different camera index in Settings > Camera
- Ensure camera is not in use by other applications
- Update camera drivers (Windows)

### Poor Detection Accuracy

- Improve lighting conditions
- Adjust camera position to eye level
- Use Calibration screen to fine-tune settings
- Try switching between detection engines

### Application Crashes

- Check logs in `~/.count-cups/logs/`
- Update dependencies: `pip install -r requirements.txt --upgrade`
- Reset settings in Settings > Advanced
- Report issue with log details

### Performance Issues

- Reduce camera resolution in Settings
- Close other applications to free resources
- Check system requirements
- Update graphics drivers
